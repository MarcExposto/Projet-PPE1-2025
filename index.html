Pourquoi ce projet ?

Nous avons réalisé ce projet dans lors du cours de Programmation et Projet Encadré du premier semestre. Pour réaliser ce projet, nous avons travaillé avec 3 langues: le français, l'anglais et le Persan. Nous avons commencé par trouver chacun 50 urls dans lequel notre mot apparaissait plusuieurs fois, le but final étant d'analyser ses contextes d'apparition. Une fois les urls récoltés, nous avons du créé un tableau, avec les imformations suivantes: une colonne pour les urls, le code http qu'elle renvoie, l'encodage de la page, son nombre total de mots, le nombre d'occurrences du mot visé, les contextes du mot, et un lien vers un concordancier et un coocurancier.

Notre mot

Nous avons choisis le mot "Confinement", "Lockdown", "Trad pe à mettre" pour son utilisation depuis la période du covid 19.
